# 🤖 ChompBot

Interactive Twitch chatbot designed to enhance viewer engagement.

---

## Overview

ChompBot automates entertainment and interaction within livestreams through dynamic chat responses and commands.

---

## Features

✅ Random jokes & pickup lines
✅ Tech facts
✅ Viewer interaction
✅ Stream customization commands
✅ Community engagement tools

---

## Systems

* Command Handler
* Event Listener
* Chat Response Engine
* Randomized Message System

---

## Built With

* Python
* Twitch API
